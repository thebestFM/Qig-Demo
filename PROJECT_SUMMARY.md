## 技术亮点

### 1. 状态管理架构
采用Pinia的Composition API风格,每个模块独立store:
```javascript
// stores/user.js - 用户健康数据
// stores/consultation.js - 问诊流程状态
// stores/prescription.js - 处方数据
// stores/companion.js - 对话历史
// stores/doctor.js - 医生列表
```

### 2. 组件化设计
- 通用组件高度可复用
- 业务组件模块化
- 样式完全scoped

### 3. 响应式布局
- 移动端优先
- 流式布局
- 横向滚动卡片

### 4. 动画系统
- CSS变量驱动
- 过渡动画流畅
- 加载状态友好

## Mock数据说明

所有数据均为演示用Mock数据:

4. **OCR识别** - 模拟的处方识别延迟
